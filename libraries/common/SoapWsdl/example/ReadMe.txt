soapIndex.php - файл для создания SOAP сервера и генерации WSDL
Ему мы скармливаем наш сервис и натравливаем на него обработчик-генератор из либы
class.soapdemo.php - пример нашего сервиса (класс, в котором у методов обязательно описаны параметры в коментариях)
Результатом работы soapIndex.php будет фвляться описание сервиса: сверху будут ссылки по которым можно посмотреть WSDL или скачать клиент для описанного сервиса
SoapDemoSoapClient.php - клиент который мы скачали после генерации WSDL для нашего сервиса
soapClientTest.php - тест клиента
